
# 项目说明

本项目为微打卡Android,IOS的客户端源码

# 使用说明

## step1:安装所需要的组件

`npm install`

`bower install`

## step2:编译www源码

`gulp build -p`

## step3:编译工程

`ionic build android`
`ionic build ios --device`
